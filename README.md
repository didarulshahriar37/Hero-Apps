# WEBSITE NAME 
HERO.IO - We Build Productive Apps
# LIVE SITE LINK
[HERO.IO](https://hero-io-d7.netlify.app/)
# FEATURES
* Trending apps are shown on the home page
* Users can install and uninstall apps.
* Users can browse all apps and installed apps with searching and sorting options.
* Each app shows detailed information including total downloads, ratings and reviews.
* Responsive design for seamless experience across all devices
